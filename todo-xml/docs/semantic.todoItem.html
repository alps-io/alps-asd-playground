Simple 'TODO' Service Profile: Semantic Descriptor

todoItem

  • type: semantic

  • tag: taxonomy

  • descriptor
    id type title
    title semantic title of the TODO item
    dueDate semantic dueDate (ISO format) of the TODO item
    category semantic free-form string identifying the category of the TODO item
    owner semantic free-form string identifying the owner (person) of the TODO item
    status semantic status of the TODO item (open or closed, defaults to open)
    list safe returns a list of TODO items
    search safe returns a filtered list (using title, category, owner) of TODO Items
    create unsafe add a new TODO item to the collection (using title, dueDate, category, owner)
    close unsafe marks an existing TODO item status as 'closed'
    update idempotent updates an existing TODO item (using dueDate, category and owner)
    remove idempotent removes an existing TODO item from the list
